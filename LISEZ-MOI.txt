NOUVEAUTÉS
- Nouvelle identité visuelle rouge et blanche.

- Toutes les fonctions de la Version 4 sont conservées : réparations, photos, commandes multi-fournisseurs, CSV et sauvegardes.

MISE À JOUR SUR GITHUB
1. Décompresser ce ZIP.
2. GitHub > dépôt de l’application > Add file > Upload files.
3. Envoyer TOUS les fichiers contenus dans ce dossier et remplacer les anciens.
4. Cliquer sur Commit changes.
5. Attendre 2 à 5 minutes.
6. Sur le téléphone, fermer complètement l’application puis la rouvrir.
7. Si l’ancienne couleur reste affichée, ouvrir l’adresse dans Chrome/Safari avec ?v=5 à la fin, puis relancer l’application.

IMPORTANT
- Les réparations déjà enregistrées restent sur l’appareil : la mise à jour ne les efface pas.
- Chaque téléphone conserve actuellement sa propre base de données. Une sauvegarde/export reste recommandée avant une mise à jour.
