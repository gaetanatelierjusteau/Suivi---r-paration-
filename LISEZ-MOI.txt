JUSTEAU SAV — VERSION 4

NOUVEAUTÉS
- Plusieurs commandes de pièces par réparation.
- Plusieurs fournisseurs possibles pour un même matériel.
- Fournisseur, référence, désignation, quantité, prix HT, dates, numéro de commande, statut et commentaire.
- Onglet Commandes global avec filtres fournisseur, statut et société.
- Export CSV des commandes.
- Reprise automatique des pièces dans le récapitulatif mensuel.

INSTALLATION
1. Décompresser le ZIP.
2. GitHub > Code > Add file > Upload files.
3. Envoyer index.html, app.js, styles.css, manifest.webmanifest, service-worker.js, icon-192.png et icon-512.png.
4. Commit changes.
5. Attendre environ 2 minutes.
6. Fermer puis rouvrir l’application. Au besoin ajouter ?v=4 à l’adresse.

Les réparations déjà présentes restent conservées sur le même appareil.
